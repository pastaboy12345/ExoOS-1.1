# ExoOS 1.1 EcoSys

ExoOS 1.1 EcoSys is a clean-reboot UNIX/DOS hybrid OS.

## Stack

- Boot protocol: Limine
- Kernel language: C
- Architecture: x86_64
- Native root volume: E:/
- Boot flow: GUI-ish boot menu -> splash screen -> prompt

## Rule

This project does not use design inspiration from the old C# ExoOS prototype.

## Build

Run:

    make iso

## Run

Run:

    make run

## Current Alpha 0.1 Goal

- Boot with Limine
- Draw a GUI-ish boot menu
- Draw a splash screen
- Enter a placeholder E:/> prompt screen
